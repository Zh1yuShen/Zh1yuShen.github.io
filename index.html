MemBuilder: Reinforcing LLMs for Long-Term Memory Construction via Attributed Dense Rewards

Preprint · Submitted to ACL 2026

Zhiyu Shen, Ziming Wu, Fuming Lai, Shaobing Lian, Yanghui Rao

We introduce MemBuilder, a reinforcement learning framework that trains a 4B-parameter model to orchestrate multi-dimensional memory construction with attributed dense rewards, outperforming Claude 4.5 Sonnet on long-term dialogue benchmarks.

HopWeaver: Synthesizing Authentic Multi-Hop Questions Across Text Corpora

Preprint · Submitted to ACL 2026

Zhiyu Shen, Jiyuan Liu, Yunhe Pang, Yanghui Rao

We propose HopWeaver, the first framework for automatically synthesizing high-quality multi-hop questions from unstructured text corpora, achieving quality on par with or exceeding human-annotated datasets.

Paper Code

CoE: A Clue of Emotion Framework for Emotion Recognition in Conversations

ACL Main 2025

Zhiyu Shen, Yunhe Pang, Yanghui Rao, Jianxing Yu

We introduce a Clue of Emotion (CoE) framework that progressively integrates multi-dimensional conversational clues with a multi-stage auxiliary learning strategy for emotion recognition in conversations.
